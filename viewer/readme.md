# Simples Werkzeug für Grundschutz++ Dokumentation
Das HTML-Dokument erlaubt es, eine Version des Kompendiums als OSCAL-Daten zu laden, den Stand der Umsetzung von Grundschutzmaßnahmen zu dokumentieren und zu speichern.

## Verwendung
Bei der Neuanlage einer Dokumentation:
1) Lade den gewünschten Stand der Grundschutzkataloge mittels Button "Kompendium".
2) Dokumentiere den Stand der Umsetzungsmaßnahmen
3) Speichere die Umsetzung mittels "CSV exportieren"

Für die Fortsetzung einer Dokumentation:
1) Lade den gewünschten Stand der Grundschutzkataloge mittels Button "Kompendium".
2) Lade die gespeicherte CSV-Datei mittels "CSV laden"
3) Dokumentiere den Stand der Umsetzungsmaßnahmen
4) Speichere die Umsetzung mittels "CSV exportieren"

Getestet mit der Version 0.9.5 vom 14.10. des GS++ Kompendiums.
